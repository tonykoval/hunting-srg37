Chapter 11 · Part III Tools & Reproducibility

Every attack in this book is packaged to re-run. Two Docker images, one baked-in catalog, a dozen scripts, and three data files. The standard of the project is that any claim here can be checked by pulling an image and running one command.

11.1   Docker images

ImageContentsUse
tonykoval/srg37-hunt Python 3.11 + numpy/networkx, pynauty, nauty CLI, cadical, kissat, cryptominisat5, all hunt scripts, the $6802$ catalog (≈ 820 MB) Every Python attack; verification
tonykoval/de-caen-filter:nauty-ccl Rust SMS+CCL two-graph engine with the C-nauty IPASIR-UP propagator Cube-and-conquer ($\S$4.4) 
docker pull tonykoval/srg37-hunt:latest
docker run --rm tonykoval/srg37-hunt:latest help        # list subcommands
# build instead: docker build -f Dockerfile.srg37-hunt -t srg37-hunt .

The images are stateless; mount a host out/ directory as a volume so results survive container exit.

The one command that matters most
# Is this candidate a NEW SRG(37,18,8,9)?
echo "<g6 string>" | docker run --rm -i tonykoval/srg37-hunt iso-compare -
#   → "DUPLICATE"   (already among the 6802)
#   → "*** NOVEL ***"   (a #6803 — verify with is_srg, then publish)

11.5   Reproduce a result

# Confirm the catalog baseline (Paley(37) is graph #1, so: DUPLICATE)
docker run --rm tonykoval/srg37-hunt iso-compare /work/data/sr371889.g6

# Re-run the fast, definitive switching negative (~140 s)
docker run --rm -v "$(pwd)/out:/work/out" tonykoval/srg37-hunt \
    gm-batch --cell-sizes 4 --max-per-size 500 --out-dir /work/out/gm

# Generate fresh unaccounted neighbour targets, then attempt extension
docker run --rm -v "$(pwd)/out:/work/out" tonykoval/srg37-hunt \
    unaccounted-extension --neighbour-g6 /work/out/unaccounted_big.g6 \
    --extension-time 1800 --max-graphs 500
A logging caveat learned the hard way

Long runs with docker run --rm piped through tee can lose all output if the daemon restarts. Use stdbuf -oL or a direct redirect to a mounted file — nine hours of compute were lost once to exactly this.

11.6   The standing invitation

The catalog, the unaccounted targets, and every attack are public. If you run path 1 or path 2 of Chapter 10 and iso-compare prints *** NOVEL ***, you have found the $6803$rd strongly regular graph on $37$ vertices. That is the whole point of writing it all down.
